◆ tstack_is_empty()

bool tstack_is_empty ( Tstack  stack)

Vérifie si une pile est vide.

Paramètres
stackLa pile à vérifier.
Renvoie
true Si la pile est vide.
false Si la pile n'est pas vide.
Note
Une pile qui ne contient tout simplement pas d'éléments, mais qui est initialisée n'est pas considérée comme vide ici.
Une pile est considérée comme vide :
- Si le contenu de la pile n'est pas initialisé (stack.s = NULL)
- Si les deux champs element_size et available_count sont nuls (=0), et le champ top = -1.
