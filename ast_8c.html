◆ ast()

Asttree ast ( Astdata_type  data_type,
  ... 
)

Retourne un arbre initialisé avec les données.

Paramètres
data_typeLe type de données.
...Les données à affecter aux données de l'arbre.
Renvoie
Asttree Un arbre initialisé avec les données.
Note
Un type de données inconnu provoquera une erreur.
Il s'agit d'une fonction variadique. Les appels de fonctions dépendront du type de données à affecter.
- Données de type constante : ast(ASTDATA_TYPE_CONST, type: <int>, constant: <int>)
- Données de type arbre : ast(ASTDATA_TYPE_TREE, type: <int>, id_lex: <int>, id_symb: <int>, son: Asttree, little_brother: Asttree)

Asttree  little_brother

Donne un petit-frère à un arbre et retourne l'arbre grand-frère.

Paramètres
treeL'arbre qui reçoit un frère.
little_brotherL'arbre petit-frère.
Renvoie
Asttree L'arbre grand-frère.
Note
L'arbre ne peut pas recevoir de petit-frère s'il est déjà un arbre grand-frère, s'il a déjà un petit-frère.
Les éléments suivants provoqueront une erreur :
- Arbre (grand-frère) vide.
- Arbre (grand-frère) a déjà un petit-frère.
