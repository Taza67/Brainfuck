\begin{DoxyItemize}
\item {\ttfamily make help} \+: affiche une notice d\textquotesingle{}utilisation du Makefile.
\item {\ttfamily make clean} \+: supprime tous les fichiers objets, tous les fichiers résiduels et tous les exécutables.
\item {\ttfamily make build} \+: compiler le programme {\bfseries{brainfuck}}.
\item {\ttfamily make rebuild} \+: recompiler le programme {\bfseries{brainfuck}}.
